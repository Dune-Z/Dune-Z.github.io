Linear Time Sequence Modeling Annotation

Have no time for this, flesh out in the future

Modeling dynamic system is the core topic in control theory. Analytical methods involve solving differential equations and require symbolic computation. In computer control, we seek to find computational models to solve the dynamic system. In SISO case, suppose that $x(t) \in \mathbb R^d$ is the state of the system at time $t$, $u(t) \in \mathbb R$ is the impulse and the response is $y(t) \in \mathbb R$. The system is described by the following equations.

$$ \begin{align} \frac{dx(t)}{dt} = F(x(t), u(t))\\ y(t) = H(x(t), u(t)) \end{align} $$

The state space is a $d$-dimensional space spanned by the state variables. And the evolution of state $x$ can be drawn as a trajectory in the state space. For example, in control, the state is usually composed of the position, velocity and acceleration of the system i.e. $x=\begin{bmatrix}r,\dot r, \ddot r\end{bmatrix}$, depending on the sensor information. We call the system a linear time-invariant (LTI) system if the function $F$ and $H$ are independent of time $t$ and linear with respect to $x(t)$ and $u(t)$. The LTI system can be described by the following equations.

$$ \begin{align} \frac{dx(t)}{dt} &= A x(t) + B u(t)\\ y(t) &= C x(t) + D u(t) \end{align} $$

The system is parameterized by $A\in\mathbb R^{d\times d}, B\in\mathbb R^{d}, C\in\mathbb R^{1\times d}$ and $D\in\mathbb R$ is the direct feedthrough term, which can be seen as an analogy to skip connection in deep neural networks. The solution to the continuous time LTI system is given by the equations below. The expressive power of the LTI system has been studied in control theory and signal processing for decades. In the context of deep learning, it is a linear recurrent neural network (RNN) and the expressive power and optimization property has been revisited.

$$ \begin{align} x(t) &= e^{At}x(0) + \int_0^t e^{A(t-\tau)}B u(\tau) d\tau\\ y(t) &= C e^{At}x(0) + \int_0^t C e^{A(t-\tau)}B u(\tau) d\tau + D u(t) \end{align} $$

Omitted the skip connection term, the solution to $y$ can be computed as a convolution in zero impulse condition. We can see that matrix $A$, also called the system matrix, heavily modulates the convolution kernel, and plays an important role in modeling the system.

$$ \begin{aligned} \mathcal K(\mu) &= C e^{A\mu}B\\ y(t) &= (\mathcal K \star u)(t) \end{aligned} $$
